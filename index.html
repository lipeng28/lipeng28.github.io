I am a staff security scientist at Baidu X-Lab. I finished my PH.D. at school of computing in University of Utah under the surpervision of Prof. Ganesh Gopalakrishnan. My research concentrates on but not limited to:

  • Bug driven hybrid fuzzing leveraging greybox fuzzing and concolic execution in C/C++

  • Dynamic analysis (symbolic/concolic execution, fuzzing) and static analysis (abstract intepretation) to improve reliability and security of complex software systems

    • Projects

    • SAVIOR: A practical bug driven hybrid fuzzing framework leveraging grepbox fuzzing and concolic executor.

    • SymJS: A symbolic executor and test case generator for JavaScript prograrms and JavaScript-based web application. I significantly extended it to scale to realistic web applications and hybrid selendroid web applications.

    • GKLEE: A symbolic analysis assisted checker and test generator for C++ CUDA programs.

    • IOC: An integer overflow checker for C/C++, integrated into Clang/LLVM as part of Undefined Behavior Sanitizer.

    • T-Check: A bug finding framework leveraging bounded model checking and random testing for sensor networks.
