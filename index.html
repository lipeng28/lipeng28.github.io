I am a staff security scientist at Baidu X-Lab. I finished my PHD at school of computing in University of Utah under the surpervision of Prof. Ganesh Gopalakrishnan. My research fields include but not limited to: application of static analysis, dynamic analysis and formal verification methodologies to improve correctness, reliability and security of large scale and complex software systems, parallel programming and computing, web technology.

Projects

  • GKLEE: A symbolic analysis assisted checker and test generator for C++ CUDA programs.

  • IOC: An integer overflow checker for C/C++.

  • TCheck: A bug finding framework for sensor networks.
